# Data Directory

This directory contains the raw datasets and annotation files used for the Breast Cancer Gene Expression Analysis project.

## Files

### GSE42568_series_matrix.txt.gz

Source: NCBI GEO (GSE42568)

Description:
Gene expression matrix containing transcriptomic profiles of breast cancer and normal breast tissue samples measured using the Affymetrix Human Genome U133 Plus 2.0 Array.

Dataset Statistics:

* Total Samples: 121
* Breast Cancer Samples: 104
* Normal Samples: 17
* Total Probes: 54,675

### GPL570.annot.gz

Source: NCBI GEO Platform GPL570

Description:
Affymetrix Human Genome U133 Plus 2.0 Array annotation file used to map probe identifiers to gene symbols, gene titles, and functional annotations.

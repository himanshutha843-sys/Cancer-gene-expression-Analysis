# Breast Cancer Gene Expression Analysis

## Overview
Transcriptomic analysis of the GSE42568 breast cancer dataset from NCBI GEO.

## Dataset
- 121 samples
- 104 breast cancer samples
- 17 normal breast tissue samples
- 54,675 gene probes

## Methods
- Data preprocessing
- Principal Component Analysis (PCA)
- Differential Expression Analysis
- Benjamini-Hochberg FDR correction
- Volcano Plot Visualization
- Heatmap Analysis

## Results
- Identified 17,922 significantly differentially expressed probes (FDR < 0.05)
- Clear separation between cancer and normal samples in PCA space
- Visualization of transcriptomic differences using PCA and volcano plots
